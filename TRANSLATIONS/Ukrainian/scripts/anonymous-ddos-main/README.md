## Установка

- Встановити [Node.JS](https://nodejs.org/)

- запустити команду `npm ci`

## Почніть

запустити команду `npm start`

## СТОП

пресовий `Ctrl + C`

## Параметри

- вимкнутий `.env` файл
- додайте значення url після `URL=`
- додайте користувацьке значення затримки після `DELAY=` за секунди. Default: `60`
- зберегти файл
